# ZenTube Chrome Extension

ZenTube is a productivity-focused Chrome Extension that removes YouTube distractions and helps users stay focused.

## Features
- Hide recommended videos
- Hide comments
- Hide YouTube Shorts
- Focus Mode toggle
- Pomodoro Timer
- Usage Tracker
- Modern dark UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extension APIs
